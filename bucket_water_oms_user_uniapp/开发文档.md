# 用户订水平台 - 开发文档

> **项目名称**: bucket_water_oms_user_uniapp  
> **框架**: UniApp (Vue 3 + TypeScript)  
> **创建日期**: 2026-05-16  
> **项目描述**: 用户端订水平台，支持选择水站、下单、查看附近水站、订单管理等功能

---

## 📋 项目概述

### 系统定位

用户订水平台是一个面向终端消费者的移动端应用，为用户提供便捷的桶装水订购服务。系统与现有的水厂订货管理系统(OMS)共享后端API，实现订单的完整流转。

### 核心功能

1. **水站发现** - 基于地理位置展示附近水站，支持按距离排序
2. **商品浏览** - 查看水站商品列表、价格、库存
3. **在线下单** - 选择商品、配送地址、支付方式完成下单
4. **订单跟踪** - 实时查看订单状态、配送进度
5. **地址管理** - 维护多个配送地址，设置默认地址
6. **用户中心** - 个人信息、订单历史、收藏水站

### 用户角色

| 角色 | 说明 |
|------|------|
| **游客** | 未登录用户，可浏览水站和商品，但不能下单 |
| **普通用户** | 注册用户，可下单、查看订单、管理地址 |
| **水站** | 商户端，管理商品、接收订单（由水站老板端处理）|

### 支持平台

> **🎯 微信小程序优先支持**，同时兼容 H5 和其他平台

| 平台 | 支持状态 | 说明 |

  - [ ] 删除水票套餐
  - [ ] 设置水票有效期
  - [ ] 设置每张水票的送水桶数

#### 水票类型说明

| 类型 | 说明 | 使用方式 |
|------|------|---------|
| **单桶购买** | 无水票，按原价购买 | 直接下单 |
| **水票套餐** | 购买水票，享受优惠 | 水票兑换下单 |

#### 水票业务规则

1. **水票设置**：每个商品可以绑定多张水票
2. **桶数灵活**：每张水票可以设置不同的送水桶数（如10桶、20桶、50桶）
3. **价格优惠**：水票价格通常低于单桶价×桶数
4. **有效期**：水票设置有效期，过期作废
5. **购买限制**：未设置水票的商品只能单桶购买
6. **水票使用**：用水票兑换时，按水票设定的桶数自动计算

### 5.5 水站配送订单模块 (station-order)

> **重要说明**: 水站配送订单是用户直接在某个水站下单后产生的订单，与仓库端的订单（`orders` 表）是**完全独立**的两套订单体系。

#### 功能列表
- [ ] **用户端**：下单、支付、查看配送进度
- [ ] **水站端**：接收订单、派单给配送员、查看配送状态
- [ ] **配送员端**：接单、开始配送、完成配送、查看配送进度
- [ ] 订单状态：待派单 → 待接单 → 配送中 → 已完成

#### 订单状态流转

```
用户下单 → 支付成功
     ↓
水站派单 → 选择配送员
     ↓
配送员接单 → 开始配送
     ↓
配送中 → 实时位置更新
     ↓
完成配送 → 用户确认
```

```sql
    price DECIMAL(10, 2) NOT NULL,
    quantity INT NOT NULL,
    subtotal DECIMAL(10, 2) NOT NULL
);

COMMENT ON TABLE station_delivery_order_item IS '水站配送订单明细表';
```

| 对比项 | 仓库订单 (orders) | 水站配送订单 (station_delivery_order) |
|--------|-------------------|--------------------------------------|

| 配送方 | 仓库配送 | 水站自配送 |
| 配送员 | 司机 (driver) | 水站配送员 (delivery_person) |
| 订单流程 | 仓库→司机→水站 | 水站→配送员→用户 |
| 派单方 | 仓库管理员 | 水站老板 |

```typescript
export const calculateDistance = (
  lat1: number,
  lon1: number,
  lat2: number,
  lon2: number
): number => {
  const R = 6371; // 地球半径（公里）
  const dLat = (lat2 - lat1) * Math.PI / 180;
  const dLon = (lon2 - lon1) * Math.PI / 180;
  const a = 
    Math.sin(dLat/2) * Math.sin(dLat/2) +
    Math.cos(lat1 * Math.PI / 180) * Math.cos(lat2 * Math.PI / 180) * 
    Math.sin(dLon/2) * Math.sin(dLon/2);
  const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
  return R * c;
};
```

### 2. Token 刷新机制

```typescript
// services/api.ts
import { getToken, setToken, refreshToken } from '@/utils/storage';

let isRefreshing = false;
let refreshSubscribers: Array<(token: string) => void> = [];

const subscribeTokenRefresh = (cb: (token: string) => void) => {
  refreshSubscribers.push(cb);
};

const onTokenRefreshed = (token: string) => {
  refreshSubscribers.forEach(cb => cb(token));
  refreshSubscribers = [];
};

export const request = async (options: RequestOptions) => {
  let token = getToken();
  
  const requestOptions = {
    ...options,
    header: {
      ...options.header,
      'Authorization': `Bearer ${token}`
    }
  };
  
  try {
    const response = await uni.request(requestOptions);
    
    if (response.statusCode === 401 && !options.url?.includes('/auth/')) {
      if (!isRefreshing) {
        isRefreshing = true;
        try {
          const newToken = await refreshToken();
          setToken(newToken);
          onTokenRefreshed(newToken);
          isRefreshing = false;
        } catch (e) {
          isRefreshing = false;
          // 跳转登录页
          uni.navigateTo({ url: '/pages/auth/login' });
          throw e;
        }
      }
      
      return new Promise((resolve) => {
        subscribeTokenRefresh(async (token) => {
          const newOptions = {
            ...options,
            header: {
              ...options.header,
              'Authorization': `Bearer ${token}`
            }
          };
          const response = await uni.request(newOptions);
          resolve(response);
        });
      });
    }
    
    return response;
  } catch (error) {
    throw error;
  }
};
```

```typescript
// stores/cart.ts
import { defineStore } from 'pinia';
import { computed } from 'vue';

export const useCartStore = defineStore('cart', () => {
  const cartItems = ref<CartItem[]>([]);
  
  const totalAmount = computed(() => 
    cartItems.value.reduce((sum, item) => sum + item.price * item.quantity, 0)
  );
  
  const totalCount = computed(() => 
    cartItems.value.reduce((sum, item) => sum + item.quantity, 0)
  );
  
  const addToCart = (product: Product, quantity: number = 1) => {
    const index = cartItems.value.findIndex(
      item => item.productId === product.id
    );
    
    if (index > -1) {
      cartItems.value[index].quantity += quantity;
    } else {
      cartItems.value.push({
        id: `cart_${Date.now()}`,
        stationId: product.stationId,
        stationName: product.stationName,
        productId: product.id,
        productName: product.name,
        price: product.price,
        quantity,
        image: product.image
      });
    }
  };
  
  const updateQuantity = (cartId: string, quantity: number) => {
    const item = cartItems.value.find(i => i.id === cartId);
    if (item) {
      if (quantity <= 0) {
        removeFromCart(cartId);
      } else {
        item.quantity = quantity;
      }
    }
  };
  
  const removeFromCart = (cartId: string) => {
    const index = cartItems.value.findIndex(i => i.id === cartId);
    if (index > -1) {
      cartItems.value.splice(index, 1);
    }
  };
  
  const clearCart = () => {
    cartItems.value = [];
  };
  
  return {
    cartItems,
    totalAmount,
    totalCount,
    addToCart,
    updateQuantity,
    removeFromCart,
    clearCart
  };
});
```

```typescript
export const calculateDistance = (
  lat1: number,
  lon1: number,
  lat2: number,
  lon2: number
): number => {
  // 使用 Haversine 公式计算两点之间的距离
  const R = 6371; // 地球半径（公里）
  const dLat = (lat2 - lat1) * Math.PI / 180;
  const dLon = (lon2 - lon1) * Math.PI / 180;
  const a = 
    Math.sin(dLat/2) * Math.sin(dLat/2) +
    Math.cos(lat1 * Math.PI / 180) * Math.cos(lat2 * Math.PI / 180) * 
    Math.sin(dLon/2) * Math.sin(dLon/2);
  const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
  return Number((R * c).toFixed(2));
};

// 检查是否在配送范围内
export const isInDeliveryRange = (
  userLat: number,
  userLon: number,
  stationLat: number,
  stationLon: number,
  deliveryRange: number
): boolean => {
  const distance = calculateDistance(userLat, userLon, stationLat, stationLon);
  return distance <= deliveryRange;
};
```

```typescript
import { request } from '@/utils/request';
import { setStorageSync, getStorageSync } from '@/utils/storage';

// 微信登录
export const wechatLogin = async (): Promise<LoginResult> => {
  try {
    // 1. 获取微信 code
    const loginRes = await new Promise((resolve, reject) => {
      uni.login({
        provider: 'weixin',
        success: (res) => {
          if (res.code) {
            resolve(res);
          } else {
            reject(new Error('获取code失败'));
          }
        },
        fail: (err) => reject(err)
      });
    });
    
    // 2. 将 code 发送到后端换取 token
    const response = await request({
      url: '/api/auth/wechat/login',
      method: 'POST',
      data: {
        code: loginRes.code
      }
    });
    
    if (response.success && response.data) {
      // 3. 保存 token 和用户信息
      setStorageSync('token', response.data.token);
      setStorageSync('userInfo', response.data.userInfo);
      setStorageSync('openId', response.data.openId);
      
      return {
        success: true,
        data: response.data
      };
    }
    
    return {
      success: false,
      message: response.message || '登录失败'
    };
  } catch (error) {
    console.error('微信登录失败', error);
    return {
      success: false,
      message: '微信登录失败，请重试'
    };
  }
};

// 获取用户手机号（需用户授权）
export const getPhoneNumber = async (encryptedData: string, iv: string): Promise<string> => {
  const response = await request({
    url: '/api/auth/wechat/decrypt-phone',
    method: 'POST',
    data: {
      encryptedData,
      iv
    }
  });
  
  if (response.success) {
    return response.data.phoneNumber;
  }
  
  throw new Error('获取手机号失败');
};
```

```typescript
export const handlePaymentFail = (error: any): string => {
  const errorCode = error.errMsg || '';
  
  if (errorCode.includes('cancel')) {
    return '用户取消支付';
  }
  
  if (errorCode.includes('fail')) {
    return '支付失败，请重试';
  }
  
  if (errorCode.includes('config_error')) {
    return '支付配置错误，请联系客服';
  }
  
  return '支付异常，请稍后重试';
};
```

```typescript
// utils/request.ts
export const handleNetworkError = (error: any): void => {
  const errorCode = error.statusCode || 0;
  
  if (errorCode === 0) {
    // 网络断开
    uni.showToast({
      title: '网络连接已断开',
      icon: 'none',
      duration: 2000
    });
  } else if (errorCode === 403) {
    // 无权限
    uni.showToast({
      title: '无访问权限',
      icon: 'none'
    });
  } else if (errorCode >= 500) {
    // 服务器错误
    uni.showToast({
      title: '服务器异常，请稍后重试',
      icon: 'none'
    });
  }
};
```
