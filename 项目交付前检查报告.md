## 六、部署与运行说明

### 6.1 后端运行

```bash
cd bucket-water-oms-java
mvn clean compile -DskipTests   # ✅ 已通过
mvn spring-boot:run              # 启动应用
# 默认端口 8080
# 数据库：PostgreSQL 192.168.31.251:5432/wateroms
```

**重要**: 启动前需执行数据库脚本：
```powershell
psql -h 192.168.31.251 -p 5432 -U wateroms -d wateroms -f "database\factory_module.sql"
psql -h 192.168.31.251 -p 5432 -U wateroms -d wateroms -f "database\platform_config.sql"
psql -h 192.168.31.251 -p 5432 -U wateroms -d wateroms -f "database\platform_operation_log.sql"
```

### 6.2 平台管理端运行

```bash
cd bucket-water-oms-platform-admin
npm install
npm run dev
# 访问 http://localhost:5173
# Vite 自动代理 /api -> http://localhost:8080
```

默认登录账号（需在数据库初始化）：
- 用户名：`platform_admin`（或任意已创建的平台管理员）
- 密码：`123456`（默认密码）

### 6.3 用户端（uniapp）运行

```bash
cd bucket_water_oms_user_uniapp
# HBuilderX 打开并运行到 H5 / 微信小程序
# 或 npm run dev:h5 / dev:mp-weixin
```

### 6.4 移动端（Flutter）运行

```bash
cd bucket_water_oms_harmony
flutter pub get
flutter run
```
