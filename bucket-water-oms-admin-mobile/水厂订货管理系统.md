##

# 水厂订货管理系统（PRD v1.1 - 已确认版）

## 一、 产品概述

### 1.1 项目背景

水厂传统销售渠道依赖电话订水+手工水卡记录，易产生错误。本系统实现水站在线下单→仓库备货派单→司机配送签收→空桶回收→仓库核对的完整闭环，所有操作均在手机端完成。

### 1.2 核心目标

- 替代电话订货，实现数字化订单流转
- 建立空桶押金与流转的精准管控
- 支持差异化销售政策（价格/账期/预存金）
- 实现全链路数据可追溯、可报表

### 1.3 系统规模

- 水站：500个
- 司机：20人
- 仓库：10个
- 并发预估：峰值50单/分钟

## 二、 角色与权限体系

| 角色    | 账号开通         | 核心权限                                 | 使用端            |
| ----- | ------------ | ------------------------------------ | -------------- |
| 水厂管理员 | 系统预设         | 开户管理、定价策略、查看全量报表、司机/仓库账号管理           | 管理后台（手机/PC自适应） |
| 仓库管理员 | 水厂管理员        | 接单/拒单、备货、派单给司机、核对空桶、处理售后             | 仓库端APP         |
| 配送司机  | 水厂管理员        | 接单、查看配送清单、配送签收（拍照）、录入回桶/欠桶、中途回仓、地图导航 | 司机端APP         |
| 水站老板  | 水厂管理员        | 查看实时库存、下订单（审查前可改）、查看历史/对账/空桶往来、账户充值  | 水站端小程序/APP     |
| 水站店员  | 水厂管理员（由老板申请） | 仅查看库存、协助收货确认（扫码/验证码）                 | 水站端小程序（受限权限）   |

**权限说明**：店员账号由水站老板向水厂申请开通，仅能看到库存数量，无法看到价格、无法下单、无法查看账户余额和欠桶数据。

## 三、 核心业务闭环流程

```
[水站端] 查看库存 → 提交订单 → （审查前可修改/取消）
     ↓
[仓库端] 接单审查 → 【库存不足：拒单退回水站】 / 【库存充足：备货】 → 手动派单给司机
     ↓
[司机端] 接单 → 地图路线规划 → 配送导航 → 到达签收（拍照+核对数量）
     ↓
     录入回桶数 + 欠桶数（如有）
     [可选] 中途回仓补货/交空桶
     ↓
[仓库端] 司机回仓 → 核对空桶实收 → 差异追责（司机承担）
     ↓
[售后] 水站发起补货/仓库发起缺货处理 → 生成售后单
     ↓
[财务] 月结/预存金扣款/信用额度控制
     ↓
[每月固定日期] 生成对账单（水站查看/导出）
```

## 四、 功能模块详细设计

### 4.1 水厂管理员后台

#### 4.1.1 水站管理

- **开户**：创建水站账号（手机号+密码），设置初始押金桶数量、信用额度
- **销售政策配置**：
  - 产品单价（每个水站独立定价，即时生效）
  - 账期类型：现结/月结/季结
  - 预存金要求金额
  - 阶梯价设置（如100桶以上单价X元）
  - 欠桶阈值（达到数量需补缴押金）
  - 空桶押金金额（按水站分别设置每桶押金金额）
- **店员账号管理**：审核水站提交的店员开通申请
- **状态管理**：启用/停用账号

#### 4.1.2 产品管理

- 桶装水规格管理（18.9L、11.3L等）
- 其他商品（瓶装水、饮水机、水泵）
- 全局库存查看（分仓库）

#### 4.1.3 司机/仓库账号管理

- 创建司机账号（姓名、电话、负责区域）
- 创建仓库账号（仓库名称、地址、覆盖区域）
- 账号启用/停用/重置密码

#### 4.1.4 报表中心

- **水站进货报表**：按月份统计每个水站各产品进货量、金额
- **司机配送报表**：按月份统计每个司机配送单数、桶数、里程
- **空桶汇总报表**：
  - 各水站欠桶数量（押金状态）
  - 各仓库空桶库存
  - 在途空桶（司机未交回）
- **财务对账报表**：
  - 应收款明细（账期客户）
  - 预存金余额明细
  - 信用额度使用情况
- **导出功能**：所有报表支持导出Excel

#### 4.1.5 对账单设置

- 固定对账日设置：每月X号（如每月1号）自动生成上月对账单
- 对账单包含：期初余额、本月进货明细、本月回款、期末应收/预存金余额

### 4.2 水站端（小程序/APP）

#### 4.2.1 权限区分（老板 vs 店员）

| 功能           | 水站老板 | 水站店员       |
| ------------ | ---- | ---------- |
| 查看实时库存       | ✓    | ✓（仅数量，无价格） |
| 下单           | ✓    | ✗          |
| 查看订单历史       | ✓    | ✗          |
| 查看对账单        | ✓    | ✗          |
| 查看空桶往来/欠桶    | ✓    | ✗          |
| 查看账户余额/信用额度  | ✓    | ✗          |
| 账户充值         | ✓    | ✗          |
| 发起售后         | ✓    | ✗          |
| 收货确认（扫码/验证码） | ✓    | ✓          |

#### 4.2.2 首页（老板视角）

- 实时库存展示（按仓库显示可订货数量）
- 快捷下单入口
- 账户余额/信用额度/欠桶数量提醒（红色预警当达到阈值）
- 对账单提醒：每月固定对账日推送新对账单

#### 4.2.3 订货流程

1. **选商品**：桶装水+其他商品，显示实时单价（按该水站政策）
2. **确认订单**：显示应付金额（根据账期政策：现结需余额充足，月结扣信用额度）
3. **提交订单**：系统检查库存+账户余额/额度
4. **订单状态跟踪**：
   - 待审查（可取消/修改）
   - 已接单备货中（不可修改）
   - 已派单（显示司机信息+电话）
   - 配送中（地图查看司机位置）
   - 已完成（显示签收照片+实收数量）
   - 已拒单（仓库无货，退回水站，需重新下单）

#### 4.2.4 我的

- **订单历史**：按时间筛选，查看详情
- **对账单**：
  - 每月固定日期自动生成上月对账单
  - 查看历史对账单（按月份筛选）
  - 导出Excel
- **空桶往来**：
  - 押金桶总数
  - 欠桶数量（红色预警当达到阈值）
  - 历史回桶记录
  - 补缴押金入口（当欠桶达到阈值时强制显示）
- **账户充值**：微信支付充值预存金

### 4.3 仓库端 (APP)

#### 4.3.1 订单管理

- **待接单列表**：显示水站订单（按距离/时间排序）
- **接单审查**：查看订单详情→确认接单/拒绝（需填原因：库存不足/其他）
- **拒单处理**：
  - 选择拒单原因（默认：库存不足）
  - 订单状态变为"已拒单"，退回水站端
  - 水站收到推送通知，需重新下单或联系水厂
- **备货中**：打印备货单（支持蓝牙打印机）
- **派单**：手动选择司机（显示司机当前任务数+位置）

#### 4.3.2 司机回仓核对

- **待核对列表**：司机提交回仓申请
- **空桶核对**：仓库清点实收空桶数 vs 司机上报数
- **差异处理**：
  - 一致：确认入库
  - 不一致：记录差异数量，系统自动标记司机承担，生成差异记录
- **打印回仓单**

#### 4.3.3 售后处理

- **售后单列表**：水站发起的补货申请
- **处理**：同意→生成新配送单 / 拒绝→填原因
- **缺货处理**：订单部分缺货时，发起缺货售后单

#### 4.3.4 库存管理

- 空桶入库（清洗后入库）
- 空桶出库（司机领用）
- 实时库存查看

### 4.4 司机端（APP - 需支持离线）

#### 4.4.1 任务中心

- **待配送列表**：按接单时间/路线优化排序
- **订单详情**：水站地址（导航）、联系人电话、商品清单、历史欠桶数
- **地图路线规划**：
  - 接单后自动规划最优配送路线（支持多订单串联配送）
  - 显示预计里程和时间
  - 支持跨城路线规划

#### 4.4.2 配送流程

1. **到达打卡**：GPS定位确认
2. **卸货核对**：与客户核对实收数量（可修改实际配送数）
3. **拍照上传**：水站门头/签收单/异常情况（至少3张）
4. **回桶录入**：
   - 本次回收空桶数
   - 欠桶数（客户无桶可退，增加欠桶）
   - 显示当前累计欠桶数
5. **客户电子签收**：
   - 方式一：客户手机签字
   - 方式二：短信验证码确认（店员可协助输入）
   - 方式三：水站老板APP确认

#### 4.4.3 中途回仓

- **回仓申请**：选择回仓仓库
- **交空桶录入**：本次交回空桶数量
- **补货申请**：领取新货
- **等待仓库核对**

#### 4.4.4 离线功能

- 配送数据本地缓存（SQLite）
- 有网络时自动同步
- 离线地图支持（需预下载区域地图，支持跨城地图包）
- 离线状态下可：查看已同步订单、录入配送数据、拍照（本地存储）

## 六、 关键业务规则

### 6.1 订单审查规则

- 水站提交后，仓库需在X小时内接单（超时提醒）
- 接单前水站可修改/取消，接单后不可变更
- 拒单规则：仓库库存不足时可拒单，选择原因后订单退回水站，状态变为"已拒单"，水站收到推送通知
- 审查时检查：库存充足性、水站账户余额/额度充足性

### 6.2 账户扣款规则

- **现结/预存金**：下单时冻结金额，配送完成后扣款
- **月结**：下单时检查信用额度，配送后计入应收，按月结算
- **欠桶阈值**：达到阈值时，水站需补缴押金才能继续下单（补缴金额=欠桶数×每桶押金金额）

### 6.3 空桶差异处理规则

- 司机回仓与仓库核对，差异=司机上报数-仓库实收数
- **差异>0（司机多报）**：按仓库实收数确认，差异部分标记为司机承担，生成赔偿记录
- **差异<0（司机少报）**：司机需说明原因，确认后承担赔偿责任，从司机工资或押金中扣除

### 6.4 价格生效规则

- 水站独立定价即时生效
- 水站下单时看到实时价格
- 已接单订单不受后续改价影响

### 6.5 对账单生成规则

- **固定日期**：每月X号（如水厂管理员设置为每月1号）自动生成上月对账单
- **生成时间**：凌晨2点自动跑批生成
- **水站收到推送通知**："您的X月份对账单已生成，请查看"
- **对账单包含**：期初余额、本月所有订单明细、本月回款记录、期末余额

## 七、 非功能性需求

### 7.1 性能要求

- 支持500水站并发，峰值50单/分钟
- 页面加载<2秒，操作响应<1秒
- 离线数据本地存储，网络恢复后自动同步
- 地图路线规划响应<3秒

### 7.2 安全要求

- 手机号+密码登录，支持短信验证码找回
- 敏感操作（改价、改政策）需二次确认
- 数据传输加密（HTTPS）
- 操作日志全记录
- 店员权限严格隔离（看不到价格和账户信息）

### 7.3 兼容性

- **水站端**：微信小程序（优先）+ iOS/Android APP
- **仓库/司机端**：iOS/Android APP（需支持离线）
- **管理后台**：PC Web + 手机自适应

### 7.4 集成需求

- 微信支付（水站充值）
- 短信通知（订单状态变更、欠桶预警、对账单生成提醒）
- 蓝牙打印（仓库备货单、回仓单）
- 地图导航（司机配送路线规划，支持跨城路线）
- 高德/百度地图SDK（路线规划、导航、距离计算）
