## 🔧 技术架构

### Clean Architecture 分层

```
┌─────────────────────────────────────┐
│         Presentation Layer          │
│  (Pages, Widgets, Components)      │
├─────────────────────────────────────┤
│          Domain Layer               │
│    (Business Logic, Use Cases)     │
├─────────────────────────────────────┤
│           Data Layer               │
│  (Repositories, Models, API)      │
└─────────────────────────────────────┘
```

### 状态管理架构

```
AppState (全局状态)
├── selectedRoleIndex
├── currentUserName
├── currentStationName
└── Methods
    ├── setRoleIndex()
    ├── setUserInfo()
    └── logout()
```
