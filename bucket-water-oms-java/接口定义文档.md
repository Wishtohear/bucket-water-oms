# 水厂订货管理系统 - 接口定义文档（RESTful API）

## 接口规范

- **基础URL**: `https://api.waterfactory.com/`
- **管理端接口**: /admin
- **司机端接口**: /driver
- **仓库端接口**: /warehouse
- **水站端接口**: /station
- **请求格式**: JSON
- **响应格式**: JSON
- **认证方式**: JWT Token (Header: `Authorization: Bearer {token}`)
- **字符编码**: UTF-8
- **时间格式**: ISO 8601 (YYYY-MM-DDTHH:mm:ssZ)

## 通用响应格式

### 成功响应

```json
{
    "code": 200,
    "message": "success",
    "data": { },
    "timestamp": "2026-04-18T10:30:00Z"
}
```

### 错误响应

```json
{
    "code": 400,
    "message": "参数错误",
    "error": "INVALID PARAMETER",
    "details": "手机号格式不正确",
    "timestamp": "2026-04-18T10:30:00Z"
}
```

## HTTP状态码

| 状态码 | 含义                    | 场景          |
| --- | --------------------- | ----------- |
| 200 | OK                    | 请求成功        |
| 201 | Created               | 创建成功        |
| 400 | Bad Request           | 参数错误        |
| 401 | Unauthorized          | 未认证/Token过期 |
| 403 | Forbidden             | 无权限访问       |
| 404 | Not Found             | 资源不存在       |
| 409 | Conflict              | 业务冲突（如库存不足） |
| 500 | Internal Server Error | 服务器内部错误     |

## 三、 订单模块 (Order)

### 1. 创建订单

**POST** `/orders`

**请求参数:**

```json
{
    "warehouse_id": "WH001",
    "product_id": "P001",
    "quantity": 50,
    "delivery_address": "桂林市秀峰区XX路XX号",
    "contact_name": "张老板",
    "contact_phone": "13800138000"
}
```

**响应数据:**

```json
{
    "code": 201,
    "data": {
        "order_id": "ORD202504180001",
        "status": "pending_review",
        "total_amount": 400.00,
        "payment_type": "monthly",
        "created_at": "2026-04-18T10:30:00Z"
    }
}
```

### 2. 获取订单列表

**GET** `/orders?status=pending_review&page=1&size=20`

**查询参数:**

| 参数          | 类型     | 说明     |
| ----------- | ------ | ------ |
| status      | string | 订单状态筛选 |
| start\_date | date   | 开始日期   |
| end\_date   | date   | 结束日期   |
| page        | int    | 页码     |
| size        | int    | 每页数量   |

**订单状态枚举:**

| 状态              | 说明          |
| --------------- | ----------- |
| pending\_review | 待审查（可修改/取消） |
| reviewed        | 已接单备货中      |
| dispatched      | 已派单         |
| delivering      | 配送中         |
| completed       | 已完成         |
| cancelled       | 已取消         |
| rejected        | 已拒单         |

## 附录：错误码对照表

| 错误码  | 说明         | 处理建议        |
| ---- | ---------- | ----------- |
| 1001 | 参数错误       | 检查请求参数格式    |

| 2004 | 余额不足       | 账户充值        |
| 2005 | 信用额度不足     | 等待额度释放或申请提升 |
| 3001 | 水站不存在      | 检查水站ID      |
| 3002 | 水站已停用      | 联系水厂管理员     |
| 4001 | 司机不存在      | 检查司机ID      |
| 4002 | 仓库不存在      | 检查仓库ID      |
| 5001 | Token过期    | 刷新Token     |
