## 业务功能对照

### 仓库端核心功能
✅ **订单管理**：接单审查 → 库存检查 → 确认接单/拒单
✅ **备货管理**：备货进度跟踪 → 打印备货单 → 派单
✅ **派单管理**：选择司机 → 查看司机状态 → 确认派单
✅ **回仓核对**：司机上报清点 → 仓库实收核对 → 差异处理
✅ **售后处理**：售后申请 → 库存检查 → 补发/退款/拒绝
✅ **库存管理**：实时库存 → 预警提示 → 盘点调整
✅ **入库管理**：采购入库 → 调拨入库 → 盘点入库
✅ **出库管理**：订单出库 → 调拨出库 → 退货出库
✅ **空桶管理**：司机回桶核验 → 清洗入库 → 司机领用出库

### 司机端核心功能
✅ **配送任务**：待配送列表 → 订单详情 → 开始配送
✅ **路线规划**：智能路线 → 导航到店 → GPS打卡
✅ **签收确认**：商品核对 → 拍照上传 → 回桶录入 → 签收方式
✅ **空桶管理**：当前欠桶 → 往来记录 → 补缴押金
✅ **对账功能**：月度对账单 → 金额明细 → 确认对账

### 水站端核心功能 ⭐ 完善
✅ **账户管理**：余额查询 → 充值（微信支付）→ 信用额度 → 欠桶押金
✅ **订货流程**：商品选购 → 价格计算 → 订单提交 → 状态跟踪 → 订单详情
✅ **空桶往来**：押金桶总数 → 欠桶数量 → 预警提醒 → 补缴押金 → 流水记录
✅ **客户管理**：客户列表 → 客户详情 → 消费统计
✅ **配送调度**：订单分派 → 司机选择 → 状态跟踪
✅ **库存管理**：实时库存 → 入库出库 → 预警提示
✅ **水票管理**：水票种类 → 购买记录 → 核销记录
✅ **数据统计**：营收趋势 → 热销排行 → 客户分析
✅ **对账单**：月度对账 → 进货明细 → 回款记录 → 确认对账 → Excel导出

## 新增页面功能详解

### 水站端新增页面

#### 1. order_detail.html - 订单详情 ⭐
**功能特性**：
- 订单状态实时显示（配送中）
- 配送进度时间线（从提交到完成）
- 司机信息展示（联系方式、评分、今日完成单数）
- 商品明细列表（名称、数量、单价）
- 支付信息（金额、支付方式）
- 空桶回收数量
- 配送地址信息
- 联系仓库和司机功能

**交互设计**：
- 触摸友好的按钮交互
- 动态进度动画
- 电话和短信快捷操作

#### 2. bucket_exchange.html - 空桶往来 ⭐
**功能特性**：
- 押金桶总数统计
- 当前欠桶数量（红色预警）
- 欠桶押金金额计算
- 欠桶阈值提醒（达到阈值限制下单）
- 押金政策展示
- 空桶流水记录（回桶、欠桶、补缴）
- 月度统计数据
- 补缴押金功能（微信支付）

**业务逻辑**：
- 欠桶达到阈值时强制显示补缴入口
- 实时计算欠桶押金
- 流水记录按类型筛选

#### 3. recharge.html - 账户充值 ⭐
**功能特性**：
- 多种充值金额选择（1,000/2,000/5,000/10,000/20,000元）
- 自定义充值金额（100-100,000元）
- 充值赠送优惠（阶梯赠送）
- 微信支付集成
- 充值明细展示
- 支付协议确认

**优惠政策**：
- 充值1,000元送10元
- 充值2,000元送30元
- 充值5,000元送100元
- 充值10,000元送250元
- 充值20,000元送600元
