## 一、 产品概述

### 1.1 项目背景

水厂传统销售渠道依赖电话订水+手工水卡记录，易产生错误。本系统实现水站在线下单→仓库备货派单→司机配送签收→空桶回收→仓库核对的完整闭环，所有操作均在手机端完成。

### 1.2 核心目标

- 替代电话订货，实现数字化订单流转
- 建立空桶押金与流转的精准管控
- 支持差异化销售政策（价格/账期/预存金）
- 实现全链路数据可追溯、可报表

### 1.3 系统规模

| 指标 | 数量 |
|------|------|
| 水站 | 500个 |
| 司机 | 20人 |
| 仓库 | 10个 |
| 并发预估 | 峰值50单/分钟 |

### 1.4 名词定义

| 术语 | 定义 |
|------|------|
| 水站 | 向终端用户（居民、企业）销售桶装水的零售店 |
| 桶装水 | 主要规格18.9L、11.3L等饮用水 |
| 空桶 | 回收的桶，需要清洗消毒后可再次使用 |
| 押金桶 | 水站购买桶时缴纳押金的桶，回收后可退押金 |
| 欠桶 | 水站应退但未退的空桶数量 |
| 预存金 | 水站预先充值的金额，用于现结订单扣款 |
| 信用额度 | 月结水站的赊账额度 |
| 对账单 | 每月固定日期生成的财务对账文件 |

## 二、 角色与权限体系

### 2.1 角色定义

| 角色 | 账号开通 | 核心权限 | 使用端 |
|------|----------|----------|--------|
| 水厂管理员 | 系统预设 | 开户管理、定价策略、查看全量报表、司机/仓库账号管理 | 管理后台（手机/PC自适应） |
| 仓库管理员 | 水厂管理员 | 接单/拒单、备货、派单给司机、核对空桶、处理售后 | 仓库端APP |
| 配送司机 | 水厂管理员 | 接单、查看配送清单、配送签收（拍照）、录入回桶/欠桶、中途回仓、地图导航 | 司机端APP |
| 水站老板 | 水厂管理员 | 查看实时库存、下订单（审查前可改）、查看历史/对账/空桶往来、账户充值 | 水站端小程序/APP |
| 水站店员 | 水厂管理员（由老板申请） | 仅查看库存、协助收货确认（扫码/验证码） | 水站端小程序（受限权限） |

### 2.2 权限说明

- **店员账号开通流程**：水站老板提交申请 → 水厂管理员审核通过 → 店员账号生效
- **店员权限限制**：仅能看到库存数量，无法看到价格、无法下单、无法查看账户余额和欠桶数据

### 3.2 流程节点详细说明

#### 3.2.1 水站下单节点

| 步骤 | 操作 | 系统检查 | 后续动作 |
|------|------|----------|----------|
| 1 | 选择商品 | - | 显示该水站的独立单价 |
| 2 | 确认数量 | - | - |
| 3 | 选择仓库 | - | 显示仓库库存数量 |
| 4 | 确认订单 | 库存检查 | 库存不足 → 提示选择其他仓库或等待 |
| 5 | - | 余额/额度检查 | 预存金不足 → 提示充值 |
| 6 | - | 欠桶阈值检查 | 超阈值 → 强制提示补缴押金 |
| 7 | 提交 | - | 生成订单，进入待审查状态 |

#### 3.2.2 仓库接单节点

| 步骤 | 操作 | 条件判断 | 后续动作 |
|------|------|----------|----------|
| 1 | 查看待接单列表 | - | 按距离/时间排序 |
| 2 | 查看订单详情 | - | 查看商品清单、水站信息 |
| 3 | 检查库存 | 库存充足 | 进入备货流程 |
| 4 | - | 库存不足 | 选择拒单原因，拒单 |
| 5 | 接单 | - | 订单状态变为"已接单" |
| 6 | 打印备货单 | - | 支持蓝牙打印机 |
| 7 | 备货完成 | - | - |
| 8 | 派单 | 选择司机 | 考虑司机当前位置、任务数 |
| 9 | - | 司机接单 | 订单状态变为"已派单" |

#### 3.2.3 司机配送节点

| 步骤 | 操作 | 输入项 | 输出结果 |
|------|------|--------|----------|
| 1 | 查看任务列表 | - | 显示待配送订单 |
| 2 | 路线规划 | - | 计算最优路线 |
| 3 | 开始配送 | - | 订单状态变为"配送中" |
| 4 | 到达打卡 | 经纬度、地址 | 记录到达时间、位置 |
| 5 | 卸货核对 | 实际配送数量 | 可调整配送数量 |
| 6 | 拍照上传 | 照片（≥3张） | 存储签收照片 |
| 7 | 回桶录入 | 回桶数量 | 记录本次回收空桶 |
| 8 | 欠桶录入 | 欠桶数量 | 记录本次欠桶 |
| 9 | 电子签收 | 签字/验证码/确认码 | 完成签收 |
| 10 | 提交配送 | - | 订单状态变为"已完成" |

#### 3.2.4 回仓核对节点

| 步骤 | 操作 | 判断条件 | 后续动作 |
|------|------|----------|----------|
| 1 | 司机申请回仓 | - | - |
| 2 | 仓库收到申请 | - | - |
| 3 | 清点实收空桶 | - | - |
| 4 | 对比 | 司机上报 = 仓库实收 | 确认入库 |
| 5 | - | 司机上报 > 仓库实收 | 差异标记司机多报，生成赔偿记录 |
| 6 | - | 司机上报 < 仓库实收 | 差异标记司机少报，要求说明原因 |
| 7 | 确认入库 | - | 更新仓库库存 |

## 七、 关键业务规则

### 7.1 订单规则

#### 7.1.1 订单创建规则

| 规则 | 说明 |
|------|------|
| 库存检查 | 下单时检查仓库库存是否充足 |
| 余额检查 | 预存金账户余额需充足 |
| 额度检查 | 信用额度需有剩余 |
| 欠桶检查 | 欠桶数量不得超过阈值 |
| 价格获取 | 使用水站独立定价 |
| 阶梯价 | 自动计算适用阶梯价 |

#### 7.1.2 订单状态流转规则

```
PENDING_REVIEW ──┬──→ CANCELLED（水站取消）
                  │
                  └──→ REVIEWED（仓库接单）──→ DISPATCHED（派单）
                                                     │
                                                     └──→ DELIVERING（开始配送）
                                                                     │
                                                                     └──→ COMPLETED（完成配送）
                                                                               ↑
REJECTED（拒单）───────────────────────────────────────────────────┘
```

#### 7.1.3 拒单规则

| 规则 | 说明 |
|------|------|
| 拒单权限 | 仅仓库可拒单 |
| 拒单原因 | 库存不足/其他原因 |
| 订单处理 | 拒单后退回水站 |
| 水站通知 | 拒单后发送通知给水站 |
| 订单状态 | 变为REJECTED |

### 7.2 扣款规则

#### 7.2.1 预存金扣款

| 时机 | 说明 |
|------|------|
| 下单时 | 不冻结，直接检查余额 |
| 配送完成时 | 扣减预存金余额 |
| 退款时 | 退还到预存金账户 |

#### 7.2.2 月结扣款

| 时机 | 说明 |
|------|------|
| 下单时 | 检查信用额度是否充足 |
| 配送完成时 | 增加已用信用额度 |
| 对账日 | 生成对账单，水站回款 |

### 7.3 空桶规则

#### 7.3.1 押金收取规则

| 时机 | 说明 |
|------|------|
| 水站开户 | 可设置初始押金桶数量 |
| 补缴押金 | 欠桶超阈值时强制补缴 |

#### 7.3.2 回桶规则

| 时机 | 说明 |
|------|------|
| 配送完成 | 司机录入回桶数量 |
| 回仓核对 | 仓库清点实收数量 |

#### 7.3.3 欠桶规则

| 规则 | 说明 |
|------|------|
| 欠桶录入 | 水站无桶可退时增加欠桶 |
| 欠桶累计 | 更新水站欠桶数量 |
| 阈值控制 | 超过阈值禁止下单 |

#### 7.3.4 差异处理规则

| 差异情况 | 处理方式 |
|----------|----------|
| 司机上报 = 仓库实收 | 正常入库 |
| 司机上报 > 仓库实收 | 多报部分司机赔偿 |
| 司机上报 < 仓库实收 | 少报部分司机赔偿 |

### 7.4 对账规则

#### 7.4.1 对账单生成规则

| 规则 | 说明 |
|------|------|
| 生成时间 | 每月固定日期凌晨2点 |
| 账期范围 | 上月1日至月末 |
| 生成内容 | 期初余额、本月明细、回款、期末余额 |
| 状态 | 默认"已生成" |

#### 7.4.2 对账单确认规则

| 规则 | 说明 |
|------|------|
| 确认权限 | 水站可确认对账单 |
| 异议处理 | 有异议可标记"有争议" |
| 确认时间 | 记录确认日期 |

    │   │

#### 8.1.2 系统检查详细

| 检查项 | 检查内容 | 失败处理 |
|--------|----------|----------|

| 额度检查（月结） | 信用额度 - 已用额度 ≥ 订单金额 | 提示额度不足，联系水厂 |
| 欠桶检查 | 欠桶数量 < 欠桶阈值 | 提示超阈值，去补缴押金 |

#### 8.4.2 差异处理详解

| 差异情况 | 差异计算 | 处理方式 | 赔偿金额计算 |
|----------|----------|----------|--------------|
| 多报 | 司机上报 - 仓库实收 > 0 | 司机赔偿多报部分 | 差异数量 × 每桶押金 |
| 少报 | 司机上报 - 仓库实收 < 0 | 司机赔偿少报部分 | 差异数量 × 每桶押金 |
