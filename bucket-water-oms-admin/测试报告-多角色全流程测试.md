## 问题发现与解决方案

### 问题 1：水站/仓库账号登录失败（历史问题）

**问题描述**:
之前测试时，使用水站联系手机号（如 13800000001）登录失败，提示"用户不存在"。

**根本原因**:
- 水站联系手机号 ≠ 水站老板账号
- 水站老板账号是独立的 `users` 表记录
- 创建水站时填写的联系电话仅用于联系方式，不作为登录账号

**解决方案**:
1. 水站老板账号：使用 `users` 表中 `role='station'` 的记录手机号
   - 13700000006 (张三)
   - 13300020003 (共和国好看)
2. 仓库管理员账号：使用 `users` 表中 `role='warehouse'` 的记录手机号
   - 13700000002 (仓库管理员A)
   - 13300003333 (鬼画符管理员)

**验证结果**: ✅ 已解决

### 问题 2：前端表单填写问题

**问题描述**:
使用 Chrome DevTools MCP 的 `fill()` 方法填写表单时，Vue 的 v-model 绑定未正确更新。

**解决方案**:
1. 使用 `type_text()` 方法代替 `fill()` 方法
2. 或使用 JavaScript 直接操作 DOM 并触发 input 事件

**验证结果**: ✅ 已解决
